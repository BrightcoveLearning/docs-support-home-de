--- title: Bild-Transcoder ---

Verwenden des App Cloud-Bildtranscodierungsdienstes, der die Leistung Ihrer App erheblich verbessern kann, indem Bilder für die mobile Bereitstellung optimiert werden.

App Cloud bietet einen Dienst, mit dem Sie Bilder für Ihre Apps transkodieren können. Mit dem Bildtranskodierungsdienst können Sie Ihr Quellbild nehmen und es in die Größe und das Format konvertieren, die Sie für Ihre Apps benötigen. Beispiel: Sie haben ein Originalbild mit 300 x 300 Pixeln, Ihre App muss es jedoch mit 100 x 100 anzeigen. Sie können den Bildtranscodierungsdienst verwenden, um sicherzustellen, dass das Bild immer die richtige Größe für seinen Kontext hat, ohne für jeden Kontext, in dem Sie das Bild verwenden, eine neue Bilddatei erstellen zu müssen. Sie können das Bild auch zuschneiden und die Qualität (Komprimierungsverhältnis) für JPG-Bilder festlegen. Mit dem Bildtranskodierungsdienst können Sie:

  • Verbessern Sie die Leistung Ihrer App deutlich, indem Sie die Dateigröße und die Netzwerkbandbreite, die zum Anzeigen Ihrer App erforderlich sind, erheblich reduzieren
  • Reduzieren Sie den Speicherbedarf, indem Sie die Größe der Bilder reduzieren.
  • Stellen Sie sicher, dass Ihre Bilder die richtige Größe haben, um auf mobilen Geräten angezeigt zu werden.
  • Nutzen Sie den App Cloud-Server-Cache, verbessern Sie die App-Leistung und reduzieren Sie die Belastung Ihrer eigenen Server.

Der App Cloud-Bildtranscodierungsdienst fungiert als Proxy zwischen Ihren Bildern und Ihren Anwendungen. Der Zugriff auf den Dienst erfolgt über eine einfache URL, in der Sie Ihrer Bild-URL unsere Transkodierungsdienst-URL voranstellen. Dadurch können Sie unseren Transcoding-Service mit sehr geringem Aufwand in Ihre Apps einbinden. Wenn Sie beispielsweise ein Bild haben, das bei . lebt http://my.example.com/image.png Normalerweise würdest du es so anfordern:

  <img  src="http://my.example.com/image.png"  alt="Bandbreiten-hungriges Bild" width="500" height="500" />

Um es in ein kleineres Bild mit einer Größe von 100 x 100 Pixeln zu transcodieren, geben Sie einfach die folgende Anfrage ein:

  <img  src="http://transcode.appcloud.brightcove.com?image=http://my.example.com/image.png&width=100&height=100"  alt="Mein Svelte-Bild" width="100" height="100" />

So verkleinern Sie ein Bild auf 100 Pixel Breite unter Beibehaltung des ursprünglichen Seitenverhältnisses:

  <img  src="http://transcode.appcloud.brightcove.com?image=http://my.example.com/image.png&width=100" width="100" />

So verkleinern Sie ein Bild auf 100 Pixel Breite oder 100 Pixel Höhe, je nachdem, welcher Wert größer ist, und behalten Sie dabei das ursprüngliche Seitenverhältnis bei:

  <img  src="http://transcode.appcloud.brightcove.com?image=http://my.example.com/image.png&max_dimension=100" />

Um das Bild zuzuschneiden und dabei ein 100x100px großes Stück in der Mitte zu erhalten, stellen Sie einfach die folgende Anfrage:

  <img  src="http://transcode.appcloud.brightcove.com?image=http://my.example.com/image.png&crop=200,200,100,100"  alt="Mein Svelte-Bild" width="100" height="100" />

Um den Bildtranscodierungsdienst nutzen zu können, muss Ihr Bild unter einer öffentlich zugänglichen URL zugänglich sein. Das Bild muss im PNG-, JPEG- oder GIF-Format vorliegen.

Abfrageparameter

Der Dienstaufruf zur Bildtranscodierung verwendet die folgenden Abfrageparameter:

Bild (erforderlich)

Die öffentlich zugängliche URL, unter der Ihr Originalbild zu finden ist. Zum Beispiel, image=http://my.example.com/image.png.

Format (optional)

Das Format des neuen Bildes, das eines sein kann png , jpg , oder gif. Wenn Sie das Format nicht angeben, bleibt das Ausgabeformat vom Originalbild unverändert.

Größenparameter (optional)

Sie können einen der Größenparameter verwenden, Breite , Höhe , oder max_dimension , wenn das neue Bild eine andere Größe als das Originalbild haben soll. Diese Parameter geben eine Dimension des endgültigen Bildes in Pixeln an.

  • Wenn Sie keine Größenparameter angeben, hat das neue Bild dieselbe Breite und Höhe wie das Originalbild.
  • Wenn Sie angeben Breite , aber nicht Höhe , hat das neue Bild die von Ihnen angegebene Breite und die Höhe, die durch das Seitenverhältnis des Originalbilds bestimmt wird.
  • Wenn Sie angeben Höhe , aber nicht Breite , hat das neue Bild die von Ihnen angegebene Höhe und die Breite, die durch das Seitenverhältnis des Originalbilds bestimmt wird.
  • Wenn Sie angeben max_dimension , dann hat die größere Dimension des neuen Bildes die von Ihnen angegebene Größe, und die kleinere Dimension hat die Größe, die durch das Seitenverhältnis des Originalbilds bestimmt wird.

In jedem Fall hat das neue Bild dasselbe Seitenverhältnis wie das Originalbild, es sei denn, Sie geben beide a Breite und Höhe und dieses Seitenverhältnis unterscheidet sich vom ursprünglichen Seitenverhältnis.

Zuschneideparameter (optional)

Verwenden einer Transcodierungs-Hilfsfunktion

Um Ihr Bild zuzuschneiden, fügen Sie den Parameter hinzu:

Zuschneiden=x,y,w,h

Hier sind x und y die Koordinaten des Startpunkts, gemessen in Pixeln von der oberen linken Ecke (0,0). w und h sind zu bewahrende horizontale und vertikale Pixel, beginnend am Startpunkt und weiter nach rechts und unten.

Wenn sowohl Beschneiden als auch Größe angegeben sind, wird zuerst das Beschneiden angewendet und dann die Größe des beschnittenen Bildes.

Qualitätsparameter (optional)

Geben Sie nur für JPG-Bilder das Komprimierungsverhältnis für das resultierende Bild an, indem Sie den Parameter hinzufügen:

Qualität=x

x ist das anzuwendende Kompressionsverhältnis; der Standardwert ist 70.

Hier ist eine praktische JavaScript-Funktion zum Generieren einer Bildtranscodierungs-URL:

  Funktion getTranscodeURL(URL, Optionen) { var src = "http://transcode.appcloud.brightcove.com?image=" + url; if (options) { if (options.max_dimension) { src += "&max_dimension=" + options.max; } anders { if (options.width) { src += "&width=" + options.width; } if (Optionen.Höhe) { src += "&height=" + options.height; } } if (optionen.format) { src += "&format=" + options.format; } if (options.crop) { src += "&crop=" + options.crop; } if (Optionen.Qualität) { src += "&quality" + options.quality; } } src zurückgeben; };

So transcodieren Sie beispielsweise ein Bild auf eine Breite von 200 Pixeln:

  var tx_url = getTranscodeURL("http://www.example.com/example.jpg", { width: 200 });

So beschneiden Sie ein Bild auf 160 x 90, beginnend mit 100 Pixeln von links und 70 Pixel von oben und geben eine Komprimierungsrate von 65 an:

var tx_url = getTranscodeURL("http://www.example.com/example.jpg", { crop: "100,70,160,90" , quality: 65 });
